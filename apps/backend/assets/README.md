# Assets Directory

Place your company branding assets here:

## Logo
- **Filename:** `logo.png`
- **Format:** PNG with transparent background
- **Recommended size:** 300x100px
- **Max file size:** 500KB

## Letterhead (Optional)
- **Filename:** `letterhead.png`
- **Format:** PNG
- **Size:** 612x792px (US Letter)
- **Usage:** Full page letterhead template

## Notes
- Files in this directory are gitignored by default
- Add your custom files locally for development
- In production, mount as Kubernetes secrets or use environment variables

## Example Structure
```
apps/backend/assets/
├── README.md
├── logo.png          # Your company logo
└── letterhead.png    # Optional full letterhead
```
